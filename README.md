# chip8-emulator

Simple CHIP-8 CPU emulator. Can currently run the IBM Logo rom and the test opcode rom.

## Install and run

```shell
cargo install
```

```shell
cargo run .\roms\test\test_opcode.ch8
```

or

```shell
cargo run .\roms\games\Airplane.ch8
```

## tests

```shell
cargo test
```
